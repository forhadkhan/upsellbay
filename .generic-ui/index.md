This directory contains guides and example UI implementations for both WordPress and WooCommerce.

**File Naming Convention**

* WordPress example files use the `wp_` prefix.
* WooCommerce example files use the `wc_` prefix.

**Example Files Index**
The following is an index of all available example files:

* [WooCommerce Backbone Confirmation Modal](wc_confirmation_modal.md) - A pattern for using the native `$.fn.WCBackboneModal` system for confirmation workflows, delete actions, and revoking access.
